## 3行でいうと

- homenet は「自宅のネットワーク機器（ルーター・NAS・PC・スマート家電など）の情報をブラウザで一覧・編集できる**自分専用の台帳**」です。
- パソコン1台で動かす想定で、インターネットには公開しません（家の中だけ）。
- `docker compose up -d --build` の **1コマンド**で起動し、ブラウザで `http://localhost:8080` を開けば使えます。
